# Adarsh Pradeep - Portfolio Website

A modern, responsive portfolio showcasing my experience, projects, skills, and certifications as an AI & App Developer.

## 🌐 Live Demo

Visit the portfolio at: [adarsh-portfolio](https://github.com/Adxrsh-17/adarsh-portfolio)

## ✨ Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Dark & Light Mode** - Toggle between themes for optimal viewing
- **Smooth Animations** - GSAP-powered animations and smooth scrolling with Lenis
- **Dynamic Content** - Portfolio data loaded from JSON
- **Resume Download** - Quick access to download resume from navbar
- **Social Links** - Direct links to LinkedIn, GitHub, and email

## 📁 Project Structure

```
├── index.html              # Main portfolio page (HTML + CSS + JS)
├── portfolio-data.json     # Content data (experience, projects, skills, etc.)
├── ADARSH_PRADEEP_RESUME.pdf  # Resume file
├── LICENSE                 # License
└── README.md              # This file
```

## 🚀 Getting Started

### Local Development

Run a local server on port 5500:

```bash
python -m http.server 5500
```

Then open `http://localhost:5500` in your browser.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Animations:** GSAP 3.12.5 + ScrollTrigger
- **Smooth Scroll:** Lenis
- **Icons:** FontAwesome 6.5.1
- **Fonts:** Google Fonts (Inter)
- **Data Format:** JSON

## 🎨 Design

- **Primary Color:** Pink (#C2185B)
- **Typography:** Inter font family
- **Layout:** CSS Grid & Flexbox
- **Breakpoints:** Responsive at 768px for mobile/tablet
